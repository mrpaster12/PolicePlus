==========================================
1. معرفی پلاگین
==========================================
این پلاگین یک سیستم کامل پلیس/وانتد/زندان را به‌همراه سیستم دستبند پیشرفته برای سرورهای Minecraft ارائه می‌دهد.

ویژگی‌های اصلی:
✅ سیستم وانتد (Wanted) با سطوح مختلف
✅ سیستم زندان با محدودیت‌های حرکتی
✅ سیستم دستبند فیزیکی و دستوری
✅ GUI پیشرفته برای مدیریت پلیس
✅ سیستم قطب‌نما برای دنبال‌کردن بازیکنان
✅ پشتیبانی از زبان فارسی و انگلیسی
✅ سازگار با Spigot/Paper 1.17+
✅ سازگار با Bukkit 1.17+
✅ تست شده روی Paper 1.20.4
✅ پشتیبانی از PlaceholderAPI

==========================================
2. نصب و راه‌اندازی
==========================================
مراحل نصب:
1. فایل police-job-1.0.0.jar را در پوشه plugins قرار دهید
2. سرور را ری‌استارت کنید
3. فایل‌های config.yml و languages/fa.yml به‌طور خودکار ایجاد می‌شوند
4. (اختیاری) PlaceholderAPI را نصب کنید

نکات مهم:
- حداقل نسخه Spigot: 1.17+
- حافظه مورد نیاز: حداقل 512MB
- پشتیبانی از UTF-8 برای زبان فارسی

⚠️ محدودیت‌های سیستم:
- حداکثر 100 زندان همزمان
- حداکثر 1000 بازیکن وانتد
- حداکثر فاصله قطب‌نما: 10,000 بلاک

📦 نصب PlaceholderAPI (اختیاری):
1. دانلود از: https://www.spigotmc.org/resources/placeholderapi.6245/
2. قرار دادن در پوشه plugins
3. ری‌استارت سرور
4. پلاگین به‌طور خودکار placeholders را ثبت می‌کند

3. تنظیمات کامل (config.yml)
==========================================

# تنظیمات اصلی
language: "fa"  # زبان: fa (فارسی) یا en (انگلیسی)

# تنظیمات وانتد
wanted:
  max_wanted_level: 5          # حداکثر سطح وانتد
  wanted_on_kill: true         # اضافه‌کردن وانتد در صورت کشتن بازیکن
  wanted_level_per_kill: 1     # مقدار افزوده‌شونده برای هر کشتن
  reset_on_death:
    enabled: false             # ریست وانتد در صورت مرگ
    max_level_to_reset: 3      # حداکثر سطح برای ریست

# تنظیمات زندان
jail:
  jail_time_per_wanted: 5      # دقیقه به‌ازای هر سطح وانتد
  max_jail_time: 60            # سقف زمان زندان (دقیقه)
  arrest_distance: 10          # حداکثر فاصله برای دستگیری (بلاک)
  clear_wanted_on_release: true # پاک شدن وانتد پس از آزادی
  default_jail_radius: 10      # شعاع محدودیت اطراف زندان

# تنظیمات قطب‌نما
compass:
  update_interval: 20          # فاصله به‌روزرسانی (تیک)؛ 20≈1 ثانیه
  max_distance: 10000          # حداکثر فاصله نمایش

# تنظیمات دستبند
handcuff:
  name: "&cدستبند"             # نام نمایشی آیتم
  lore: "&7برای دستگیر کردن بازیکنان"  # توضیحات آیتم
  name_color: "&a"             # رنگ نام آیتم
  name_bold: true              # بولد بودن نام
  max_time: 300                # حداکثر زمان دستبند (ثانیه)
  accept_plain_item: true      # پذیرش آیتم ساده (بدون نام)

# تنظیمات نمایش
display:
  show_wanted_stars: true      # نمایش ستاره‌های وانتد
  show_wanted_level: true      # نمایش سطح وانتد
  tablist_format: "&c{player} &7[&c{level}&7]"  # فرمت Tablist

4. دستورات کامل (Commands)
==========================================

📋 دستورات وانتد (/wanted):
/wanted [self]                 # مشاهده سطح وانتد خود
/wanted set <player> [level]   # تنظیم سطح وانتد بازیکن
/wanted list                   # نمایش لیست بازیکنان وانتد
/wanted jail <player>          # زندانی‌کردن بر اساس سطح وانتد
/wanted unjail <player>        # آزاد کردن بازیکن از زندان
/wanted reload                 # ریلود تنظیمات/زبان
/wanted help                   # راهنمای کامل
/wanted gui                    # رابط گرافیکی لیست وانتد


📋 دستورات زندان (/jail):
/jail create <name> <id>       # ساخت زندان در مکان فعلی
/jail delete <name>            # حذف زندان
/jail spawn                    # تنظیم اسپاون عمومی برای زندان‌ها
/jail list                     # نمایش لیست زندان‌ها
/jail help                     # راهنمای زندان

📋 دستورات پلیس (/police):
/police                        # باز کردن GUI پلیس
/police jail                   # زندانی کردن بازیکن دستگیر شده
/police arrest <player>        # دستگیری دستی (بدون دستبند)

📋 دستورات قطب‌نما (/compass):
/compass <player>              # دریافت قطب‌نما برای دنبال‌کردن بازیکن

📋 دستورات دستبند:
/cuffe give <player> [amount]  # دادن آیتم دستبند به بازیکن
/cuffe <player>                # دستبند زدن با دستور
/cuffe                         # دستبند زدن نزدیک‌ترین بازیکن (≤3 بلاک)
/uncuffe <player>              # بازکردن دستبند

 مثال‌های کاربردی:
- تنظیم وانتد: /wanted set PlayerName 3
- زندانی کردن: /wanted jail PlayerName
- دستبند زدن: /cuffe PlayerName
- دنبال کردن: /compass PlayerName

==========================================
نکات مهم درباره ریلود و زبان (به‌روز)
==========================================

- دستور /wanted debug حذف شده است.
- دستور /wanted reload اکنون علاوه‌بر تنظیمات اصلی، تنظیمات دستبند (handcuff) را هم ریلود می‌کند.
- تغییر متریال/نام/لور دستبند فقط روی آیتم‌های جدید اثر می‌گذارد. برای دریافت آیتم جدید از /cuffe give استفاده کنید.
- اگر می‌خواهید فقط دستبندهای «امضادار» (با تگ داخلی) کار کنند و آیتم ساده با همان متریال عمل نکند، در config.yml مقدار handcuff.accept_plain_item را false بگذارید.
- سیستم زبان به‌صورت پیش‌فرض از en.yml داخلی به‌عنوان دیفالت استفاده می‌کند و کلیدهای ناقص را به فایل زبان انتخابی شما نیز می‌نویسد. در صورت قدیمی بودن فایل‌های زبان در سرور، پوشه plugins/PolicePlugin/languages را پاک کنید و سرور را ری‌استارت کنید تا نسخه‌های جدید تولید شوند.

🔐 پرمیشن‌های دستبند:

policeplugin.handcuff.give     # دادن آیتم دستبند (پیش‌فرض: op)
policeplugin.handcuff.uncuff_any # بازکردن دستبند هر بازیکن (پیش‌فرض: op)

6. سیستم دستبند (Handcuff System)
==========================================

🎯 نحوه کار:
- آیتم دستبند ثابت است (BLAZE_ROD) و از کانفیگ تغییر نمی‌کند؛ فقط نام و لور قابل تغییرند.
- فاصله دستبند زدن: حداکثر 3 بلاک
- مدت زمان دستبند: قابل تنظیم (پیش‌فرض 300 ثانیه)

🎯 اثرات دستبند:
✅ سرعت راه‌رفتن/پرواز صفر می‌شود
✅ بازیکن پشت پلیس نگه داشته می‌شود
✅ نمی‌تواند بلوک بگذارد/بشکند
✅ نمی‌تواند حرکت مستقل داشته باشد
✅ وضعیت "arrested" ثبت می‌شود

🎯 روش‌های دستبند زدن:
1. دستوری: /cuffe <player>
2. فیزیکی: راست‌کلیک با آیتم دستبند
3. خودکار: /cuffe (نزدیک‌ترین بازیکن)

🎯 آزاد کردن دستبند:
- دستوری: /uncuffe <player>
- خودکار: پس از زندانی کردن
- تایم‌اوت: پس از max_time ثانیه

🎯 برگشت دستبند:
- هنگام زندانی کردن، دستبند به پلیس برمی‌گردد
- اگر انبار پلیس پر باشد، روی زمین می‌افتد

7. سیستم زندان (Jail System)
==========================================

🏛️ ویژگی‌های زندان:
✅ محدودیت حرکتی در شعاع مشخص
✅ عدم امکان شکستن/گذاشتن بلوک
✅ نمایش زمان باقیمانده در Action Bar
✅ آزادی خودکار پس از اتمام زمان
✅ پاک شدن وانتد پس از آزادی (قابل تنظیم)

🏛️ مدیریت زندان:
- ایجاد زندان: /jail create <name> <id>
- حذف زندان: /jail delete <name>
- تنظیم اسپاون: /jail spawn
- لیست زندان‌ها: /jail list

🏛️ زندانی کردن:
- بر اساس سطح وانتد: /wanted jail <player>
- از طریق GUI: کلیک روی بازیکن در GUI پلیس
- زمان زندان: سطح وانتد × jail_time_per_wanted

🏛️ آزاد کردن:
- دستوری: /wanted unjail <player>
- خودکار: پس از اتمام زمان
- انتقال به اسپاون: در صورت تنظیم

10. PlaceholderAPI
==========================================

📊 پلیس‌هولدرهای موجود:
%police_wanted_level%          # سطح وانتد بازیکن
%police_wanted_stars%          # ستاره‌های وانتد
%police_is_wanted%             # آیا وانتد است (true/false)
%police_is_jailed%             # آیا زندانی است (true/false)
%police_jail_time_remaining%   # زمان باقیمانده زندان

📊 نحوه استفاده:
- در Scoreboard
- در Tablist
- در Chat
- در سایر پلاگین‌ها

 مثال Scoreboard با PlaceholderAPI:
Title: &6&lپلیس سرور
Line 1: &7وانتد: &c%police_wanted_level%
Line 2: &7ستاره‌ها: &e%police_wanted_stars%
Line 3: &7زندان: &4%police_is_jailed%

13. نکات مهم
==========================================

⚠️ نکات امنیتی:
- پرمیشن‌ها را با دقت تنظیم کنید
- از bypass فقط برای ادمین‌ها استفاده کنید
- فایل‌های config را در جای امن نگه دارید
- از /wanted reload فقط در صورت نیاز استفاده کنید
- پرمیشن‌ها را در گروه‌های جداگانه تنظیم کنید
- لاگ‌های دسترسی را بررسی کنید

⚠️ نکات عملکردی:
- update_interval قطب‌نما را زیاد نکنید
- تعداد زندان‌ها را محدود کنید
- از max_wanted_level معقول استفاده کنید

⚡ بهینه‌سازی عملکرد:
- compass.update_interval: 20 (پیش‌فرض مناسب)
- display.enabled: true (فقط در صورت نیاز)
- jail.default_jail_radius: 10 (کمتر = عملکرد بهتر)

⚠️ نکات سازگاری:
- با WorldGuard سازگار است
- با EssentialsX سازگار است
- با PlaceholderAPI سازگار است

⚠️ نکات پشتیبانی:
- از فایل‌های پشتیبان استفاده کنید
- تغییرات را در فایل‌های جداگانه تست کنید
- لاگ‌ها را بررسی کنید

🔍 لاگ‌های مهم:
- [PolicePlugin] INFO: Language file loaded successfully
- [PolicePlugin] WARNING: Message not found: xxx
- [PolicePlugin] ERROR: Failed to load language file
